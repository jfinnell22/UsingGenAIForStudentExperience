# Using Generative AI to Create Meaningful Student Experiences

Welcome! This site accompanies my Faculty for Undergraduate Neuroscience (FUN) presentation and workshop on AI-supported learning in neuroscience education.

The goal of this workshop is not to ask, "How can AI complete an assignment?" Instead, it asks:

> How can AI help us create learning experiences that were previously difficult, impractical, or impossible to provide at scale?

The examples and resources shared here are grounded in a learning principles-first approach. Effective AI-supported activities begin with clear learning goals, identify the intellectual work students should perform, and then leverage AI to enhance—not replace—student thinking.

### Design Principles
1. Identify learning goals and outcomes.
2. Determine constraints and boundaries.
3. Design the learning experience.
4. Use AI to generate meaningful interactions.
5. Scaffold student thinking.
6. Refine and iterate based on evidence of learning.

## Featured Question

**Who is doing the intellectual work?**

The most effective AI-supported activities preserve student ownership of reasoning, decision-making, interpretation, and reflection.
